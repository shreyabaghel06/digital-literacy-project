## 📌 Project Overview

This repository is my **Digital Literacy Portfolio**, submitted as part of the CSE0001 course project at VIT Bhopal University. I have taken on the role of a **Student Digital Ambassador**, with the goal of helping peers navigate the digital world responsibly and effectively.

The project covers **five tasks**, each mapped to a course module:
Task 1 - Create a Digital Literacy Awareness Infographic
Task 2 - Build Your Student Digital Portfolio
Task 3 - Explore Coding & Collaboration Platforms
Task 4 - Draft a Professional Email & Etiquette Guide
Task 5 - Cybercrime Awareness Case Study & Prevention Guide

All work is documented in a written project report inside the `report/` folder.
